# 🔧 AI Predictive Maintenance Dashboard

An AI-based system that predicts machine health and Remaining Useful Life (RUL) using machine learning models and visualizes results through an interactive dashboard.


## 🚀 Features

* Predict Remaining Useful Life (RUL)
* Classify machines into:

  * ✅ Healthy
  * ⚠️ Warning
  * 🚨 Critical
* Interactive dashboard using Streamlit
* Engine-wise performance analysis
* Alert monitoring system


## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Matplotlib
* Streamlit
* Machine Learning (RUL Prediction Model)


## 📁 Project Structure

```
src/                # Core logic
app.py              # Streamlit dashboard              
outputs/            # Processed results
requirements.txt
README.md
.gitignore
```


## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the application:

```
streamlit run app.py
```
